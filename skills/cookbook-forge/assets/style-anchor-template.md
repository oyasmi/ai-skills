# 风格锚点（内部文档 · 非成书内容）

> 这份文件不是给读者读的，是给**每个写作 turn 开头重新注入**的一页纸。一本几万字、
> 跨几十个 turn 写成的书，术语、比喻、口吻会不知不觉漂移——写到第 25 章时，上下文里
> 早就没有第 2 章定下的那些决策了。除非每章动笔前都回到这一页对齐。
>
> 保持它**短**。只放「跨章必须一致」的东西，别塞事实（事实在 `grounding.md`）。
> 写作阶段随书就地更新：某个术语首次定稿、某招式用掉一次配额，都回来记一笔。
> 用法：删掉本说明与示例行后再填。

---

## 1. 冻结术语表（最先冻结，之后只增不改）

一个概念全书只用**一个**译法、一个定义。别第 3 章叫「证据」、第 9 章叫「异常证据片段」、
第 18 章叫「结构化证据」。要改译法就全局改，不能局部漂。

| 核心术语（原文 / 概念） | 选定中文译法 | 一句话定义 | 首次定稿章节 |
|---|---|---|---|
| （示例）Evidence | 证据 | <一句话，与正文首次定义处一致> | p1-02 |
|  |  |  |  |

## 2. 贯穿比喻（如果这本书有一个）

- **主比喻**：<例：编织 / 急诊分诊 / 流水线>。它映射到主题的哪几个部件？逐一列出，避免比喻跑偏。
- **用法纪律**：点到为止，不要每段都套（见 `writing-craft.md` §2 ④）。宁可某章不用，也不硬塞。

## 3. 口吻三条（每章开写前默念）

- <例：像在白板前跟同事讲，不写论文摘要、不写政府报告>
- <例：善用破折号（——）做插入与转折，制造节奏>
- <例：能给具体数字就不用「多个 / 显著 / 大量」>

## 4. Callout 配额 & 破形账本（防每章雷同）

- 每章 callout ≤ 3：`tip`=正向指引 / `warn`=坑 / `info`=旁注与交叉引用。
- **破形章记账**（授权见 `chapter-blueprint.md`「五段式是默认，不是牢笼」）：
  案例复盘 __ 次 · FAQ __ 次 · 决策索引 __ 次——一本大书各用 2–4 次，别扎堆也别一次不用。
